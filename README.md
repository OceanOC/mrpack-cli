## Getting Started
### Installing
```go install github.com/oceanoc/mrpack-cli@latest```
### Building
#### Dependencies
[Golang](https://go.dev)
#### Steps
1. `git clone https://github.com/OceanOC/mrpack-cli.git`
2. `go build`
3. `./mrpack-cli`
